## Create/Update Vehicle Certification

<RequestExample>
```bash cURL (Create)
curl -X POST https://api.mantis.com/api/vehicles/cert_vehicle/ \
  -H "Content-Type: application/json" \
  -d '{
    "vehicle_id": 1,
    "name": "SOAT",
    "date_start": "2024-01-01",
    "date_end": "2025-01-01",
    "description": "Annual SOAT certification"
  }'
```

```bash cURL (Update)
curl -X PUT https://api.mantis.com/api/vehicles/cert_vehicle/ \
  -H "Content-Type: application/json" \
  -d '{
    "id": 1,
    "vehicle_id": 1,
    "name": "SOAT",
    "date_start": "2024-01-01",
    "date_end": "2025-06-01",
    "description": "Updated SOAT certification"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "message": "Certificación de vehículo creada exitosamente",
  "data": {
    "id": 1,
    "vehicle_id": 1,
    "vehicle_plate": "ABC-123",
    "name": "SOAT",
    "name_display": "SOAT",
    "date_start": "2024-01-01",
    "date_end": "2025-01-01",
    "description": "Annual SOAT certification",
    "created_at": "2024-01-01T10:00:00Z",
    "updated_at": "2024-01-01T10:00:00Z"
  }
}
```
</ResponseExample>

Create a new vehicle certification or update an existing one.

### Endpoint
```
POST /api/vehicles/cert_vehicle/
PUT /api/vehicles/cert_vehicle/
```

### Request Body

<ParamField path="id" type="integer">
  Certification ID (required for PUT/update only)
</ParamField>

<ParamField path="vehicle_id" type="integer" required>
  ID of the vehicle to certify
</ParamField>

<ParamField path="name" type="string" required>
  Certification name/type
</ParamField>

<ParamField path="date_start" type="string" required>
  Certification start date in YYYY-MM-DD format
</ParamField>

<ParamField path="date_end" type="string" required>
  Certification end date in YYYY-MM-DD format (must be after date_start)
</ParamField>

<ParamField path="description" type="string">
  Optional certification description
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="message" type="string">
  Success or error message
</ResponseField>

<ResponseField name="data" type="object">
  Created or updated certification object
  
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Certification ID
    </ResponseField>
    
    <ResponseField name="vehicle_id" type="integer">
      Associated vehicle ID
    </ResponseField>
    
    <ResponseField name="vehicle_plate" type="string">
      Vehicle license plate
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Certification name
    </ResponseField>
    
    <ResponseField name="name_display" type="string">
      Human-readable certification name
    </ResponseField>
    
    <ResponseField name="date_start" type="string">
      Certification start date
    </ResponseField>
    
    <ResponseField name="date_end" type="string">
      Certification end date
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Certification description
    </ResponseField>
  </Expandable>
</ResponseField>

## Delete Vehicle Certification

<RequestExample>
```bash cURL (Delete single)
curl -X DELETE https://api.mantis.com/api/vehicles/cert_vehicle/1/
```

```bash cURL (Delete multiple)
curl -X POST https://api.mantis.com/api/vehicles/cert_vehicle/1/ \
  -H "Content-Type: application/json" \
  -d '{"ids": [1, 2, 3]}'
```
</RequestExample>

<ResponseExample>
```json Response (Single)
{
  "success": true,
  "message": "Certificación de vehículo eliminada exitosamente",
  "data": {
    "id": 1,
    "vehicle_plate": "ABC-123",
    "name": "SOAT",
    "name_display": "SOAT"
  }
}
```

```json Response (Multiple)
{
  "success": true,
  "message": "3 certificaciones eliminadas exitosamente",
  "data": {
    "deleted": [
      {"id": 1, "vehicle_plate": "ABC-123", "name": "SOAT", "name_display": "SOAT"},
      {"id": 2, "vehicle_plate": "ABC-123", "name": "REVISION", "name_display": "Revision Tecnica"}
    ],
    "errors": [],
    "total_deleted": 2,
    "total_errors": 0
  }
}
```
</ResponseExample>

Soft delete vehicle certification(s). Single deletion uses DELETE method with ID in URL, bulk deletion uses POST with IDs in request body.

### Endpoint
```
DELETE /api/vehicles/cert_vehicle/{pk}/
POST /api/vehicles/cert_vehicle/{pk}/
```

### Path Parameters

<ParamField path="pk" type="integer">
  Certification ID to delete (for single deletion)
</ParamField>

### Request Body (for bulk deletion)

<ParamField path="ids" type="array">
  Array of certification IDs to delete
</ParamField>
