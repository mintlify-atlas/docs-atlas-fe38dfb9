## Get Sheet Items

Retrieve all items and custody chains for a specific work order sheet.

<CodeGroup>
```bash cURL

  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

## Update Sheet Detail Days

Update the billing days (monthdays_apply_cost) for a specific sheet detail item. This automatically recalculates quantity and totals.

<CodeGroup>
```bash cURL
curl -X PUT https://api.mantis.com/api/workorders/sheets/detail/5/days/ \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "monthdays_apply_cost": [1, 2, 3, 5, 8, 9, 10, 12, 15, 16, 17, 19, 22, 23, 24, 26, 29, 30]
  }'
```
</CodeGroup>

### Path Parameters

<ParamField path="detail_id" type="integer" required>
  ID of the sheet detail to update
</ParamField>

### Request Body

<ParamField body="monthdays_apply_cost" type="array" required>
  Array of day numbers (1-31) to apply billing charges. Each day represents one unit of billing for EQUIPO type resources.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the update was successful
</ResponseField>

<ResponseField name="message" type="string">
  Success or error message
</ResponseField>

<ResponseField name="data" type="object">
  Updated detail information
  
  <Expandable title="Data Object">
    <ResponseField name="id" type="integer">
      Detail item ID
    </ResponseField>
    
    <ResponseField name="monthdays_apply_cost" type="array">
      Array of billing days (sorted and deduplicated)
    </ResponseField>
    
    <ResponseField name="quantity" type="number">
      Automatically calculated quantity (number of selected days)
    </ResponseField>
    
    <ResponseField name="unit_price" type="number">
      Unit price per day
    </ResponseField>
    
    <ResponseField name="total_line" type="number">
      Total line amount (quantity × unit_price)
    </ResponseField>
    
    <ResponseField name="sheet_totals" type="object">
      Recalculated sheet totals
      
      <Expandable title="Sheet Totals">
        <ResponseField name="subtotal" type="number">
          Updated subtotal for the entire sheet
        </ResponseField>
        
        <ResponseField name="tax_amount" type="number">
          Updated tax amount (15% of subtotal)
        </ResponseField>
        
        <ResponseField name="total" type="number">
          Updated total (subtotal + tax)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

### Validation Rules

- Cannot update details for closed sheets (`is_closed = true`)
- Cannot update details if sheet status is `INVOICED` or `CANCELLED`
- Only valid day numbers between 1 and 31 are accepted
- Invalid days are filtered out automatically
- Duplicate days are removed automatically
- The quantity is automatically set to the count of selected days
- Sheet totals are automatically recalculated after the update

### Automatic Calculations

When days are updated:
1. `quantity` = number of days in `monthdays_apply_cost` array
2. `total_line` = `quantity` × `unit_price`
3. Sheet `subtotal` = sum of all detail `total_line` values
4. Sheet `tax_amount` = `subtotal` × 0.15 (15% tax)
5. Sheet `total` = `subtotal` + `tax_amount`

### Use Case

This endpoint is typically used for EQUIPO (equipment) type resources where billing is calculated based on the number of days the equipment was used during the billing period. For example:

- Equipment rented for 18 days in January
- Select days: [1, 2, 3, 5, 8, 9, 10, 12, 15, 16, 17, 19, 22, 23, 24, 26, 29, 30]
- Quantity automatically becomes 18
- If unit_price is $150, total_line = 18 × $150 = $2,700
